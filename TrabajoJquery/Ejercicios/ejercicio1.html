Ejercicio 1 Best Practices

Escribe un documento HTML que utilice las siguientes buenas prácticas relativas a la carga de las librerías de Jquery:

• Usar un CDN

• Implementar un retroceso para que se cargue el fichero Jquery local en caso de no estar disponible el del CDN.

• No usar protocolo (http o https) a la hora de enlazar la librería.

• Usar la última versión.
